## Installation
```bash
sudo apt install rdesktop
```

## Recommand Usage
```bash
rdesktop -bg 800x920 hostname.com -r disk:share=/home/directory/share
```

- -b : 비트맵 캐시 활성화
- -g : 해상도 설정
- -r : 공유 디렉토리 설정
