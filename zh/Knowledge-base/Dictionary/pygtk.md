# 概述
PyGTK 是 Python 编程语言的图形用户界面 (GUI) 库。它是一个开源库，为 GTK+ GUI 库提供面向对象的接口。 PyGTK 用于为用 Python 编写的应用程序创建图形用户界面。它是一个跨平台库，这意味着它可以在多种操作系统上使用，例如 Linux、Windows 和 Mac OS X。PyGTK 是创建图形用户界面的强大工具，在开源社区中得到广泛使用.
