# 使用 TensorFlow.js 和 Node.js 进行预测

在本文中，我们将学习如何在 Node.js 环境中使用 TensorFlow.js 模型进行预测。我们将使用预先训练的模型对图像进行预测，然后我们将训练模型对新数据进行预测。

## 先决条件

在我们开始之前，您需要做一些事情才能跟进：

* Node.js 安装在你的机器上
* 文本编辑器 - 我们推荐 [Visual Studio Code](https://code.visualstudio.com/)
* 对 JavaScript 的基本理解
* 对机器学习的基本了解

如果您在安装 Node.js 时需要帮助，请查看此 [教程](https://www.tensorflow.org/js/tutorials/setup)。

## 使用预训练模型进行预测

我们将从加载预训练模型开始。对于此示例，我们将使用经过训练的模型对手写数字的图像进行分类。

首先，我们需要安装“@tensorflow/tfjs-node”包。我们可以通过运行以下命令从命令行执行此操作：

```
npm install @tensorflow/tfjs-node
```

安装包后，我们可以在代码中导入它：

```javascript
const tf = require('@tensorflow/tfjs-node');
```

现在我们准备好加载模型了。我们将使用 `tf.loadLayersModel()` 函数从 URL 加载模型。我们可以从[TensorFlow.js model zoo](https://github.com/tensorflow/tfjs-models/tree/master/handwriting-recognition)获取模型URL。

```javascript
const modelUrl = 'https://storage.googleapis.com/tfjs-models/tfjs/mnist_model/model.json';

const model = tf.loadLayersModel(modelUrl);
```
