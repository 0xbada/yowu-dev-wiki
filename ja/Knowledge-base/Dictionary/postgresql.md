# 例
次の例は、PostgreSQL でデータベースを作成する方法を示しています。

まず、データベースを作成します。

```
CREATE DATABASE mydatabase;
```

次に、テーブルを作成します。

```
CREATE TABLE mytable (
  id INTEGER PRIMARY KEY,
  name VARCHAR(50)
);
```

最後に、いくつかのデータをテーブルに挿入します。

```
INSERT INTO mytable (id, name) VALUES (1, 'John');
```
