# Hacer predicciones con TensorFlow.js y Node.js

En esta publicación, aprenderemos cómo hacer predicciones con un modelo TensorFlow.js en un entorno Node.js. Usaremos un modelo previamente entrenado para hacer predicciones sobre imágenes y luego entrenaremos un modelo para hacer predicciones sobre nuevos datos.

## Requisitos previos

Antes de comenzar, hay algunas cosas que deberá tener para seguir:

* Node.js instalado en su máquina
* Un editor de texto: recomendamos [Visual Studio Code] (https://code.visualstudio.com/)
* Una comprensión básica de JavaScript
* Una comprensión básica del aprendizaje automático

Si necesita ayuda para instalar Node.js, consulte este [tutorial](https://www.tensorflow.org/js/tutorials/setup).

## Hacer predicciones con un modelo previamente entrenado

Comenzaremos cargando un modelo previamente entrenado. Para este ejemplo, usaremos un modelo que ha sido entrenado para clasificar imágenes de dígitos escritos a mano.

Primero, necesitaremos instalar el paquete `@tensorflow/tfjs-node`. Podemos hacer esto desde la línea de comando ejecutando:

```
npm install @tensorflow/tfjs-node
```

Una vez instalado el paquete, podemos importarlo en nuestro código:

```javascript
const tf = require('@tensorflow/tfjs-node');
```

Ahora estamos listos para cargar el modelo. Usaremos la función `tf.loadLayersModel()` para cargar el modelo desde una URL. Podemos obtener la URL del modelo del [zoológico modelo de TensorFlow.js](https://github.com/tensorflow/tfjs-models/tree/master/handwriting-recognition).

```javascript
const modelUrl = 'https://storage.googleapis.com/tfjs-models/tfjs/mnist_model/model.json';

const model = tf.loadLayersModel(modelUrl);
```
