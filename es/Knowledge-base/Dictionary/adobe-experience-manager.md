# Descripción general
Adobe Experience Manager (AEM) es un sistema de gestión de contenido (CMS) desarrollado por Adobe que permite a las empresas crear, gestionar y ofrecer experiencias digitales en sitios web, aplicaciones móviles y formularios. # Descripción
Adobe Experience Manager es una plataforma integral para crear, administrar y entregar experiencias digitales. Combina un sistema de administración de contenido (CMS) con un sistema de administración de activos digitales (DAM) para permitir que las empresas creen, administren y brinden experiencias digitales en sitios web, aplicaciones móviles y formularios.

AEM proporciona una interfaz basada en web para que los autores de contenido creen, editen y publiquen contenido. También proporciona una biblioteca de componentes que se pueden utilizar para crear sitios web y aplicaciones de forma rápida y sencilla. También proporciona herramientas para crear componentes personalizados e integrarse con aplicaciones de terceros.

La plataforma está diseñada para ser extensible, lo que permite a las empresas personalizar la plataforma para satisfacer sus necesidades específicas. También proporciona un modelo de seguridad sólido para proteger el contenido y garantizar el cumplimiento de los estándares y regulaciones de la industria.

# Historia
Adobe Experience Manager se lanzó inicialmente en 2011 como un sistema de gestión de contenido (CMS) para sitios web. A lo largo de los años, ha evolucionado hasta convertirse en una plataforma integral para crear, gestionar y ofrecer experiencias digitales.

En 2015, Adobe lanzó la aplicación Experience Manager Mobile, que permitía a las empresas crear y administrar aplicaciones móviles desde la plataforma AEM. En 2017, Adobe lanzó el producto AEM Forms, que permitió a las empresas crear y administrar formularios desde la plataforma AEM.
