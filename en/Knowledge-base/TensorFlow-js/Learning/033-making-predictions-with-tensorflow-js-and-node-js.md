# Making Predictions with TensorFlow.js and Node.js

In this post, we'll learn how to make predictions with a TensorFlow.js model in a Node.js environment. We'll use a pre-trained model to make predictions on images, and then we'll train a model to make predictions on new data.

## Prerequisites

Before we get started, there are a few things you'll need to have in order to follow along:

* Node.js installed on your machine
* A text editor - we recommend [Visual Studio Code](https://code.visualstudio.com/)
* A basic understanding of JavaScript
* A basic understanding of machine learning

If you need help installing Node.js, check out this [tutorial](https://www.tensorflow.org/js/tutorials/setup).

## Making Predictions with a Pre-trained Model

We'll start by loading a pre-trained model. For this example, we'll use a model that's been trained to classify images of handwritten digits.

First, we'll need to install the `@tensorflow/tfjs-node` package. We can do this from the command line by running:

```
npm install @tensorflow/tfjs-node
```

Once the package is installed, we can import it in our code:

```javascript
const tf = require('@tensorflow/tfjs-node');
```

Now we're ready to load the model. We'll use the `tf.loadLayersModel()` function to load the model from a URL. We can get the model URL from the [TensorFlow.js model zoo](https://github.com/tensorflow/tfjs-models/tree/master/handwriting-recognition).

```javascript
const modelUrl = 'https://storage.googleapis.com/tfjs-models/tfjs/mnist_model/model.json';

const model = tf.loadLayersModel(modelUrl);
```
