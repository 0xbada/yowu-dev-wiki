# TensorFlow.js 및 Node.js로 예측하기

이 게시물에서는 Node.js 환경에서 TensorFlow.js 모델로 예측하는 방법을 알아봅니다. 미리 훈련된 모델을 사용하여 이미지를 예측한 다음 모델을 훈련시켜 새 데이터를 예측합니다.

## 전제 조건

시작하기 전에 따라하기 위해 필요한 몇 가지 사항이 있습니다.

* 컴퓨터에 설치된 Node.js
* 텍스트 편집기 - [Visual Studio Code](https://code.visualstudio.com/)를 권장합니다.
* JavaScript에 대한 기본적인 이해
* 머신러닝에 대한 기본적인 이해

Node.js 설치에 도움이 필요하면 이 [튜토리얼](https://www.tensorflow.org/js/tutorials/setup)을 확인하세요.

## 사전 훈련된 모델로 예측하기

미리 학습된 모델을 로드하는 것으로 시작하겠습니다. 이 예에서는 손으로 쓴 숫자의 이미지를 분류하도록 훈련된 모델을 사용합니다.

먼저 `@tensorflow/tfjs-node` 패키지를 설치해야 합니다. 명령줄에서 다음을 실행하여 이 작업을 수행할 수 있습니다.

```
npm install @tensorflow/tfjs-node
```

패키지가 설치되면 코드에서 가져올 수 있습니다.

```javascript
const tf = require('@tensorflow/tfjs-node');
```

이제 모델을 로드할 준비가 되었습니다. URL에서 모델을 로드하기 위해 `tf.loadLayersModel()` 함수를 사용할 것입니다. [TensorFlow.js 모델 동물원](https://github.com/tensorflow/tfjs-models/tree/master/handwriting-recognition)에서 모델 URL을 얻을 수 있습니다.

```javascript
const modelUrl = 'https://storage.googleapis.com/tfjs-models/tfjs/mnist_model/model.json';

const model = tf.loadLayersModel(modelUrl);
```

우리가 로드한 모델은 손으로 쓴 숫자의 이미지를 높은 정확도로 분류할 수 있습니다. 그러나 예측을 위해 모델을 사용하기 전에 이미지를 사전 처리해야 합니다.
