# 보안 통신을 위한 가상 사설망(VPN) 설정

VPN(가상 사설망)을 사용하면 인터넷을 통해 다른 네트워크에 안전하게 연결할 수 있습니다. VPN을 사용하여 지역 제한 웹사이트에 액세스하고, 공용 Wi-Fi에서 다른 사람의 눈에 띄지 않도록 브라우징 활동을 보호할 수 있습니다.

대부분의 인터넷 사용자는 VPN에 대해 걱정할 필요가 없습니다. Netflix를 시청하거나 검색 데이터를 암호화하려는 경우 TunnelBear와 같은 무료 VPN 서비스를 사용할 수 있습니다. 그러나 보안을 강화하거나 자체 서버에 VPN을 설정하려면 VPN 서버를 사용해야 합니다.

## VPN 서버란?

VPN 서버는 기기와 인터넷 사이의 게이트웨이 역할을 하는 컴퓨터 또는 가상 머신입니다. VPN 서버에 연결하면 모든 트래픽이 해당 서버를 통해 라우팅됩니다. 즉, ISP(인터넷 서비스 공급자)는 더 이상 사용자가 온라인에서 수행하는 작업을 볼 수 없습니다.

VPN 서버를 사용하여 지역 제한 웹사이트에 액세스하고, 공용 Wi-Fi에서 눈에 띄지 않도록 브라우징 활동을 보호할 수 있습니다.

## VPN 서버 설정 방법

VPN 서버를 설정하는 방법에는 여러 가지가 있습니다. 이 기사에서는 Windows Server 2016을 사용하여 VPN 서버를 설정하는 방법을 보여줍니다.

### 1. Windows Server 2016 설치

아직 서버가 없는 경우 Windows Server 2016을 구매하여 설치해야 합니다. 여기에서 이에 대한 지침을 찾을 수 있습니다.

### 2. 네트워크 어댑터 구성

Windows Server 2016을 설치했으면 네트워크 어댑터를 구성해야 합니다. 제어판으로 이동하여 네트워크 및 공유 센터를 열면 됩니다.

어댑터 설정 변경을 클릭한 다음 구성하려는 네트워크 어댑터를 마우스 오른쪽 버튼으로 클릭합니다. 메뉴에서 속성을 선택합니다.

속성 창에서 인터넷 프로토콜 버전 4(TCP/IPv4) 옵션을 선택하고 속성 버튼을 클릭합니다.

IPv4 속성 창에서 다음 IP 주소 사용 옵션을 선택하고 사용할 IP 주소, 서브넷 마스크 및 기본 게이트웨이를 입력합니다.

확인 버튼을 클릭하여 변경 사항을 저장합니다.

### 3. DNS 설정

다음 단계는 DNS를 설정하는 것입니다. DNS는 기억하기 어려운 IP 주소(예: 192.168.1.1) 대신 기억하기 쉬운 도메인 이름(예: www.example.com)을 사용할 수 있게 해주는 시스템입니다.

제어판으로 이동하여 DNS 관리자를 열어 서버에서 DNS를 설정할 수 있습니다.

DNS 관리자에서 DNS 서버를 마우스 오른쪽 버튼으로 클릭하고 메뉴에서 속성 옵션을 선택합니다.

DNS 서버 속성 창에서 전달자 탭을 선택합니다.

편집 버튼을 클릭하고 사용할 DNS 서버의 IP 주소를 입력합니다. 여기에서 퍼블릭 DNS 서버 목록을 찾을 수 있습니다.

확인 버튼을 클릭하여 변경 사항을 저장합니다.

### 4. 라우팅 및 원격 액세스 서비스 설치

다음 단계는 라우팅 및 원격 액세스 서비스(RRAS)를 설치하는 것입니다. RRAS는 VPN 서버를 설정할 수 있는 Microsoft 서비스입니다.

서버 관리자로 이동하고 역할 및 기능 추가 옵션을 선택하여 RRAS를 설치할 수 있습니다.

역할 및 기능 추가 마법사에서 역할 기반 또는 기능 기반 설치 옵션을 선택하고 다음 버튼을 클릭합니다.

기능 선택 페이지에서 원격 액세스 확인란을 선택하고 다음 버튼을 클릭합니다.

설치 선택 확인 페이지에서 설치 버튼을 클릭합니다.

설치가 완료되면 닫기 버튼을 클릭합니다.

### 5. RRAS 구성

이제 RRAS가 설치되었으므로 구성해야 합니다. 서버 관리자로 이동하여 도구 옵션을 선택하면 됩니다.

도구 메뉴에서 라우팅 및 원격 액세스 옵션을 선택합니다.

라우팅 및 원격 액세스 콘솔에서 서버를 마우스 오른쪽 단추로 클릭하고 라우팅 및 원격 액세스 구성 및 사용 옵션을 선택합니다.

구성 마법사에서 사용자 지정 구성 옵션을 선택하고 다음 버튼을 클릭합니다.

사용자 지정 구성 페이지에서 VPN 액세스, NAT 및 LAN 라우팅 확인란을 선택하고 다음 버튼을 클릭합니다.

요약 페이지에서 마침 버튼을 클릭합니다.

### 6. VPN 사용자 생성

다음 단계는 VPN 서버에 연결할 수 있는 사용자를 만드는 것입니다. 서버 관리자로 이동하여 도구 옵션을 선택하면 됩니다.

도구 메뉴에서 Active Directory 사용자 및 컴퓨터 옵션을 선택합니다.

Active Directory 사용자 및 컴퓨터 콘솔에서 사용자 폴더를 마우스 오른쪽 버튼으로 클릭하고 메뉴에서 새 사용자 옵션을 선택합니다.

새 사용자 창에서 사용자의 사용자 이름, 전체 이름 및 설명을 입력합니다.

다음 버튼을 클릭합니다.

비밀번호 페이지에서 사용자의 비밀번호를 입력하고 다음 버튼을 클릭합니다.

암호 확인 페이지에서 마침 버튼을 클릭합니다.

### 7. VPN 액세스 구성

다음 단계는 사용자에 대한 VPN 액세스를 구성하는 것입니다. 서버 관리자로 이동하여 도구 옵션을 선택하면 됩니다.

도구 메뉴에서 라우팅 및 원격 액세스 옵션을 선택합니다.

라우팅 및 원격 액세스 콘솔에서 서버를 확장하고 원격 액세스 정책 옵션을 클릭합니다.

원격 액세스 정책 창에서 Microsoft 라우팅 및 원격 액세스 서버에 대한 연결을 마우스 오른쪽 버튼으로 클릭하고 메뉴에서 속성 옵션을 선택합니다.

Microsoft 라우팅 및 원격 액세스 서버 속성 창에 연결에서 사용자 탭을 선택합니다.

추가 버튼을 클릭하고 VPN 서버에 연결하도록 허용할 사용자를 선택합니다.

확인 버튼을 클릭하여 변경 사항을 저장합니다.

### 8. VPN 클라이언트 구성

마지막 단계는 VPN 서버에 연결하려는 장치에서 VPN 클라이언트를 구성하는 것입니다.

다양한 VPN 클라이언트를 사용할 수 있지만 기본 제공 Windows VPN 클라이언트를 사용합니다.

VPN 클라이언트를 구성하려면 제어판으로 이동하여 네트워크 및 공유 센터를 엽니다.

새 연결 또는 네트워크 설정 옵션을 클릭합니다.

연결 또는 네트워크 설정 창에서 네트워크에 수동으로 연결 옵션을 선택하고 다음 버튼을 클릭합니다.

연결 속성 페이지에서 VPN 연결 정보를 입력합니다. 여기에는 VPN 유형(IKEv2), VPN 서버 주소, VPN 사용자 이름 및 VPN 암호가 포함됩니다.

만들기 버튼을 클릭합니다.

이제 VPN 연결이 네트워크 및 공유 센터에 나열됩니다.

VPN 서버에 연결하려면 연결을 마우스 오른쪽 버튼으로 클릭하고 메뉴에서 연결 옵션을 선택합니다.

메시지가 표시되면 VPN 사용자 이름과 암호를 입력하고 연결 버튼을 클릭합니다.

이제 VPN 서버에 연결되어야 합니다.
