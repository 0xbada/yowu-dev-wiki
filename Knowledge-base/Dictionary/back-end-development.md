# 개요
백엔드 개발은 웹 사이트 또는 애플리케이션의 백엔드 개발에 중점을 둔 소프트웨어 엔지니어링 유형입니다. 데이터베이스, API 및 서버측 로직을 포함하는 웹 애플리케이션의 서버측을 담당합니다. 백엔드 개발자는 데이터베이스, API 및 서버측 로직을 포함하는 웹사이트 또는 애플리케이션의 백엔드 개발을 담당합니다. 그들은 또한 프런트엔드와 백엔드의 통합을 담당합니다.
