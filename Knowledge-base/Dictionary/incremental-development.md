# 개요
증분 개발은 시간이 지남에 따라 시스템에 작고 점진적인 변경 사항을 제공하는 데 중점을 둔 소프트웨어 개발 방법론입니다. 이 접근 방식은 대규모의 복잡한 시스템을 개발할 때 개발 프로세스에 대한 더 많은 유연성과 제어를 허용하므로 종종 사용됩니다. 증분 개발을 사용하여 기존 시스템에 새로운 기능과 개선 사항을 신속하게 제공할 수도 있습니다.

# 역사
