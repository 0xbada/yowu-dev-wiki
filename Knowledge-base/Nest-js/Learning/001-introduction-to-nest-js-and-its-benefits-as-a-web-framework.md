# 001: Nest.js 소개 및 웹 프레임워크로서의 이점

Nest.js는 서버 측 애플리케이션을 만들기 위한 웹 프레임워크입니다. TypeScript로 작성되었으며 내부적으로 Express.js를 사용합니다.

Nest.js는 서버 측 애플리케이션을 만드는 데 탁월한 선택입니다. 사용하기 쉽고 배우기 쉽습니다. Nest.js 프레임워크는 매우 다재다능하며 다양한 애플리케이션에 사용할 수 있습니다.

다음은 Nest.js 사용의 이점 중 일부입니다.

- 사용하기 쉽고 배우기 쉽습니다.
- Nest.js는 매우 다재다능하며 다양한 애플리케이션에 사용할 수 있습니다.
- Nest.js는 매우 빠르고 효율적입니다.
- Nest.js는 매우 확장 가능합니다.
- Nest.js에는 훌륭한 문서가 있습니다.

서버 측 애플리케이션을 만들기 위한 웹 프레임워크를 찾고 있다면 Nest.js가 탁월한 선택입니다.
